## Operaciones con Listas en Python

Este programa implementa funciones básicas para trabajar con listas de números.

## Funcionamiento:

- **Duplicar valores**: Multiplica cada elemento de la lista por 2.
- **Máximo**: Obtiene el valor mayor de la lista.
- **Promedio**: Calcula el promedio de los elementos.

- duplicar(lista) → Devuelve una nueva lista con los valores duplicados.
- maximo(lista) → Devuelve el número mayor de la lista.
- promedio(lista) → Devuelve el promedio de los elementos.
- main() → Ejecuta pruebas con distintas listas.
